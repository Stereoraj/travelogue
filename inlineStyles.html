Inline Style

We can apply CSS in a single element by inline CSS technique.
The inline CSS is also a method to insert style sheets in HTML document. This method mitigates some advantages of style sheets so it is advised to use this method sparingly.
If you want to use inline CSS, you should use the stle attribute to the relevant tag.

Syntax: 
			<htmltag style="cssproperty1:value; cssproperty2:value;">
			</htmltag>

limitations of inline styling

  • Redundant code: As the style is scoped to the element, you cannot reuse the code, you should be placing the code at various places
  • Maintainance of the code is difficult as you have to update at various places if any change is anticipated.
  • You cannot extend the scope of the style through classes and different types of elements.
